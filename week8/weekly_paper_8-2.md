# 🛒 쇼핑몰 구매 퍼널 이벤트 설계

## 1. 이벤트 정의 개요

상품 상세 페이지 → 장바구니 추가 → 결제 완료의 사용자 행동 흐름을 분석하기 위해
핵심 이벤트를 정의하고 각 이벤트에서 수집해야 할 주요 속성을 설계한다.

---

## 2. 이벤트 설계

### 2.1 상품 상세 조회

* **이벤트명:** product_view

* **설명:**
  사용자가 특정 상품 상세 페이지에 진입했을 때 발생하는 이벤트

* **수집 속성:**

  * product_id: 상품 고유 ID
  * category: 상품 카테고리
  * price: 상품 가격

---

### 2.2 장바구니 추가

* **이벤트명:** add_to_cart

* **설명:**
  사용자가 상품을 장바구니에 추가했을 때 발생하는 이벤트

* **수집 속성:**

  * product_id: 상품 고유 ID
  * quantity: 선택한 수량

---

### 2.3 결제 완료

* **이벤트명:** purchase

* **설명:**
  사용자가 결제를 완료하여 구매가 확정되었을 때 발생하는 이벤트

* **수집 속성:**

  * order_id: 주문 고유 ID
  * total_amount: 총 결제 금액
  * payment_method: 결제 수단

---

## 3. 설계 의의

* 상품 조회 → 장바구니 → 구매까지의 전환율 분석
* 특정 상품 및 카테고리별 구매 성과 분석
* 가격, 결제 수단 등 다양한 요인이 구매에 미치는 영향 분석
